# 주차장 관리 프로그램

### 고급C프로그래밍 개인 프로젝트

## 개요
자료구조 중 하나인 LinkedList를 이용하여 자유로운 주제로 프로그램을 만들어보자.

## 개발 언어 및 환경
* Visual Studio
* C language

## 내용
자동차의 번호판 형식(ex) 12가3456)을 이용하여 차의 종류와 번호를 가지고 추가, 삭제, 갱신, 검색 등의 기능을 구현하고

파일 입출력을 이용하여 현재 주차 현황을 저장 및 불러오는 기능을 구현하였다.
